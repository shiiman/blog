---
description: ブログ記事のアイキャッチ画像を生成する
---

1. 対象となる記事のパス（例: `drafts/2026-01-03_my-article/article.md`）を確認します。
2. 記事本文を読み取り、タイトル・主要キーワード・想定読者・トーンを抽出します。
3. 抽出結果をもとに、以下のデザインルールを満たす画像生成プロンプトを作成します。
   - **アスペクト比**: 16:9（1280x720相当）
   - **ベースカラー**: 白（`#FFFFFF`）
   - **アクセントカラー**: 青（`#007BFF`）と濃いグレー（`#424242`）
   - **テキスト**: 記事タイトルを中央に大きく配置（太字サンセリフ）
   - **スタイル**: テクニカル・クリーン（清潔感、構造的、視認性重視）
4. `generate_image` ツールで画像を 1 枚生成します。
   - 推奨: `aspect_ratio=16:9`, `output_mime_type=image/png`
5. 記事ディレクトリ配下に `assets/` を作成し、生成画像を `assets/eyecatch.png` として保存します。
6. 生成結果を確認し、必要に応じてプロンプトを調整して再生成します。

※ `wp-cli post` / `wp-cli update` は `assets/eyecatch.png` が存在する場合、自動で WordPress にアップロードしてアイキャッチへ設定します。既存画像を差し替える場合は `wp-cli update <file> --force-eyecatch` を使用します。
